# i3 workspaces

  - markdown: vim + firefox
  - thunderbird
  - gimp
  - notes file (drop down)
  - music
    - youtube music
  - communications
    - slack
    - facebook
    - whatsapp
  - password manager (drop down)

## Shortcuts
$mod + z --> workspace mode (more shortcuts for each workspace)
$mod + ?x? --> last urgent workspace-window
